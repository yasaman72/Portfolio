Make a Doll

February 2020

a clymation game made with real play doughs. It's an experimental game in which you use the oddly-shaped body parts to make a doll with another player. The game was made for the Global Game Jam.

  • Game Engine: Unity
  • My Role: Game Designer, Visual Designer

I used the dough to make the shapes, scanned them, cleaned the scans and imported them into the game. I also deformed the scans to use them for the wobbling animation.
